# djang_RMS
基于Django的后台管理系统

## 1.介绍
- 前后端不分离，UI框架使用Bootstrap，使用Django自带的模板语法
- 表单数据校验使用Django自带的ModelForm

## 2.支持功能
- 多表与单表的增删改查
- 数据分析图表展示，使用echarts
- 将数据导出为excel文件，使用Pandas
- 自定义的分页组件
